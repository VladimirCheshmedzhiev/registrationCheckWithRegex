# registrationCheckWithRegex

A code which checks if the registration which is input is valid. 
there are conditions in which the registration is valid. 
In order to be valid the input has to be made of both username and password, however:
the username must be divided by U$ on both ends and the password - "P@$". 
another condition specifically for the password is that it must contain atleast three numbers.

All conditions can be modified using regular expressions in the conditionals inside the the loop.
